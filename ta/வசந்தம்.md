# வசந்தம்

வசந்தம் மீண்டும் வந்துவிட்டது.

நிலத்தில் பனி நீராக மாறியது, மற்றும் நீர் கால்வாய்களில் ஓடியது. பச்சை மொட்டுகள் கிளைகளிலிருந்து வெளிப்படுகின்றன, வெட்கமும் பயந்தும், உலகத்தைப் பார்ப்பதற்கு பயந்தது போல. காற்று வெப்பமாகிவிட்டது, ஆனால் முகத்தில் வீசும்போது, இன்னும் மண்ணின் மண் வாசனையை கொண்டு வருகிறது.

விழுங்குகள் திரும்பி வந்துள்ளன, கூரைகளின் கீழ் கீச்சுகின்றன. வயல்களில், விவசாயிகள் குனிந்து, விதைகளை மண்ணில் தூவுகின்றனர் — அவர்கள் இலையுதிர்காலத்தை நம்புகிறார்கள், ஆனால் இலையுதிர்காலம் உண்மையில் அவர்களின் ஆசைகளை நிறைவேற்றுமா?

நான் எப்போதும் மோசமான தீய எண்ணத்துடன் இயற்கையைப் பற்றி யூகிக்க தயாராக இருந்தேன், ஆனாலும் வசந்தம் வந்துவிட்டது. நம்பிக்கை அல்லது நிராசை, வாழ்க்கை தொடர வேண்டும்.
