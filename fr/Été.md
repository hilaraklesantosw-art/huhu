# Été

L'été est accablant.

Les cigales crient désespérément dans les arbres, comme si elles mourraient si elles cessaient. La lumière du soleil est poison, brûlant tout. L'ombre sous les arbres offre un soulagement, mais ce soulagement n'est que temporaire. Les fleurs de lotus dans l'étang s'épanouissent dans toute leur gloire, mais combien s'arrêtent vraiment pour les contempler ?

Les enfants jouent dans l'eau ; les aînés s'assoient sous les arbres, s'éventant avec des éventails de palmier. La nuit, les lucioles clignotent, comme la dernière lumière restante en ce monde.

Certains disent que l'été est passionné. Moi, je trouve cette passion bien trop intense — elle laisse haletant. Quand la vie est à son plus vigoureux, elle est souvent aussi la plus proche de la décrépitude.
