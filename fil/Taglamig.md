# Taglamig

Ang taglamig ay tahimik.

Ang niyebe ay bumabagsak, tinatakpan ang lahat. Ang lupa ay puti, ang mga puno ay puti, kahit ang mga bubungan ay puti. Ang puting ito ay dalisay, ngunit malamig at walang awa rin. Ang mga yelo ay nakabitin sa ilalim ng mga bubungan, kristal na malinaw, ngunit nagpapalabas ng lamig.

Ang mga tao ay nagtitipon sa paligid ng pugon, umiinom ng mainit na tsaa, nagsasalita ng mga bagay-bagay. Ang mga bata ay naglalaro sa niyebe — may tawanan, oo, ngunit gaano katagal ang tawanang iyon?

Ang taglamig ay panahon ng paninirahan — hindi ako naniniwala sa mga magagandang salitang iyon. Taglamig ay taglamig; lamig ay lamig. Walang dapat pagandahan. Paano ang paghihintay sa tagsibol? At kung ang tagsibol ay dumating? Ito ay isa pang siklo lamang.
