# Taglagas

Ang taglagas ay dumating, at ang mga dahon ay naging dilaw.

Ang palay ay hinahog, ibinababa ang ulo parang sumuko sa kapalaran. Ang mga mansanas ay nakabitin na pula at lila mula sa mga sanga, naghihintay na pitasin. Ang mga dahon ng maple ay pula, ang ginkgo dilaw — maganda, oo, ngunit ano ang darating pagkatapos ng kagandahan? Wala kundi pagkabulok sa putik.

Ang mga ganso ay lumilipad patimog, sumisigaw, parang nagpapaalam, o baka humihikbi. Ang buwan ng Gitnang Taglagas ay puno, ngunit ang mga tao ay hindi kinakailangan ay muling nagkakatipon.

Sinasabi nila na ang taglagas ay panahon ng ani. Gayunpaman, pagkatapos ng ani ay ang pamumutlay. Pasasalamat? Sa halip ay iniisip ko na sa mundong ito, ang karamihan ng mga bagay ay dapat ibayad pabalik.
