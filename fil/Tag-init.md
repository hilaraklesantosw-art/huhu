# Tag-init

Ang tag-init ay mainit na mainit.

Ang mga veve ay sumisigaw nang desperado sa mga puno, parang mamamatay kung titigil. Ang liwanag ng araw ay nakakalason, sinusunog ang lahat. Ang anino sa ilalim ng mga puno ay nagbibigay ng ginhawa, ngunit ang ginhawang iyon ay pansamantala lamang. Ang mga bulaklak ng lotus sa lawa ay namumulaklak sa buong kaluwalhatian, ngunit ilang tao ba ang talagang humihinto upang tingnan ang mga ito?

Ang mga bata ay naglalaro sa tubig; ang mga matatanda ay nakaupo sa ilalim ng mga puno, nagpapahangin gamit ang mga dahon ng palma. Sa gabi, ang mga alitaptap ay kumikislap, parang huling liwanag na natitira sa mundong ito.

Ang ilan ay nagsasabi na ang tag-init ay mapusok. Ako, gayunpaman, natatagpuan ang passion na ito ay masyadong matindi — iniwan ka nitong humihingal. Kapag ang buhay ay nasa pinakamasigla nitong sandali, madalas ito ay pinakamalapit din sa pagkasira.
