# Iarnă

Iarna este tăcută.

Zăpada cade, acoperind totul. Pământul este alb, copacii sunt albi, chiar și acoperișurile sunt albe. Această albețe este pură, dar și rece și implacabilă. Țurțurii de gheață atârnă sub streașină, clari ca cristalul, dar emanând frig.

Oamenii se adună în jurul sobei, beau ceai fierbinte, vorbesc despre lucruri triviale. Copiii se joacă în zăpadă — există râs, da, dar cât timp poate dura un astfel de râs?

Iarna este un sezon de așezare — nu am crezut niciodată în acele cuvinte frumoase. Iarnă este iarnă; frig este frig. Nu este nimic de înfrumusețat. Cât despre așteptarea primăverii? Și dacă primăvara vine? Este doar un alt ciclu.
